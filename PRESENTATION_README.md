## 🎨 Customization Tips

### For PowerPoint Slides

**Color Scheme Suggestions:**
- SQL Agent references: Blue (#2196F3)
- RAG Agent references: Green (#4CAF50)
- Both/Synthesis: Orange (#FF9800)
- Background: White or light gray
- Code blocks: Dark theme (VS Code style)

**Fonts:**
- Headings: Calibri, Arial, or Helvetica Bold
- Body: Calibri, Arial, or Helvetica Regular
- Code: Consolas, Monaco, or Courier New

**Layout:**
- Use consistent header/footer
- Slide numbers in corner
- Logo/school name if required
- Leave room for speaker notes

### For Diagrams

**Converting Mermaid to Images:**
1. Visit https://mermaid.live
2. Copy diagram code from PRESENTATION_DIAGRAMS.md
3. Paste into left panel
4. Preview appears on right
5. Click "Actions" → "PNG" or "SVG"
6. Download and insert into slides

**Or use GitHub:**
1. Create a temporary .md file with diagram
2. Push to GitHub
3. GitHub renders it automatically
4. Screenshot the rendered diagram

## 🚨 Troubleshooting

### "I don't have time to create slides"
**Solution:** Use PRESENTATION.md as speaking notes, show diagrams in markdown, do mostly live demo

### "I'm nervous about live demo"
**Solution:** Record a demo video beforehand as backup, practice demo 5+ times

### "Question I don't know answer to"
**Solution:** "Great question! Let me check the code..." (open PRESENTATION.md or actual code)

### "Running out of time"
**Solution:** Skip CLI demo and code walkthrough, focus on web demo and architecture

### "Technical issue during demo"
**Solution:** Have screenshots ready, or show recorded video, or walk through code instead
